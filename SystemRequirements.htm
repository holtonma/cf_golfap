The Aviary is a  cutting edge golf software tool and therefore has the 
following PC SYSTEM REQUIREMENTS:

  - Windows XP, Windows 2000, or Windows ME
  - 300 MHz processor or higher
  - 64 MB RAM or higher (128 recommended)
  - Hard disk drive with 200 MB free disk space
  - 4x speed CD-ROM drive recommended
  - 1024x768 resolution monitor
  - Sound board and speakers recommended for some features
  - Works with any printer supported by Windows XP, Windows 2000 or Windows ME
